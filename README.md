# Nous - مكرر النصوص الذكي 🚀

**Nous** هو أداة ويب متطورة وبسيطة مصممة لتكرار النصوص وتنسيقها بسرعة فائقة مع ميزات ذكية وتكامل حي مع قاعدة بيانات Firebase.

### 🔗 رابط المعاينة الحية
[https://kemo-omar.github.io/Nous/](https://kemo-omar.github.io/Nous/)

---

## ✨ المميزات (Features)

- **تكرار فائق السرعة:** كرر أي نص مئات أو آلاف المرات بضغطة زر.
- **خيارات تنسيق متعددة:**
  - إضافة ترقيم تلقائي للسطور.
  - تحويل الحروف الإنجليزية إلى كبيرة (Uppercase).
  - ميزة عكس النص (Reverse Text) لإضافة لمسات إبداعية.
- **فواصل مخصصة:** اختر الفاصل بين الجمل (سطر جديد، مسافة، رموز مثل `-` أو `*`).
- **تكامل حي مع Firebase:** - عداد عالمي يعرض إجمالي عدد الحروف التي تم توليدها بواسطة جميع مستخدمي الموقع لحظياً.
- **تجربة مستخدم عصرية (UX/UI):**
  - وضع ليلي (Dark Mode) جذاب ومريح للعين.
  - زر نسخ سريع مع تنبيه (Toast Notification).
  - زر مشاركة مباشرة للنص المكرر عبر واتساب.
- **عداد حروف محلي:** لمتابعة حجم النص الناتج قبل استخدامه.

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **Frontend:** HTML5, CSS3 (Modern UI/UX).
- **Logic:** JavaScript (Vanilla JS).
- **Backend:** Firebase Firestore (Real-time Database).
- **Fonts:** Google Fonts (Cairo).
